## Настройте вашу среду разработки

Узнайте, как обновлять вашу документацию локально и публиковать ее.

### Редактирование и предварительный просмотр

<AccordionGroup>
  <Accordion icon="github" title="Клонируйте вашу документацию локально">
    В процессе подключения мы создали репозиторий на вашем Github с
    содержимым вашей документации. Вы можете найти этот репозиторий в нашей[панели управления](https://dashboard.mintlify.com). Чтобы клонировать репозиторий
    локально, следуйте этим[инструкциям](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    в вашем терминале.
  </Accordion>

  <Accordion icon="rectangle-terminal" title="Предварительный просмотр изменений">
    Предварительный просмотр помогает убедиться, что ваши изменения выглядят так, как задумано. Мы создали
    интерфейс командной строки для локального рендеринга этих изменений.

    1. Установите[Mintlify CLI](https://www.npmjs.com/package/mintlify) для предварительного просмотра
       изменений документации локально с помощью этой команды:`npm i -g mintlify`
    2. Выполните следующую команду в корневой директории вашей документации (где находится`docs.json`):`mintlify dev`

    <Note>
      Если вы в настоящее время используете устаревший`mint.json`конфигурационный файл, пожалуйста, обновите Mintlify CLI:

      `npm i -g mintlify@latest`
      И запустите новую команду обновления в вашем репозитории документации:

      `mintlify upgrade`
      Теперь вы должны использовать новый`docs.json`конфигурационный файл. Не стесняйтесь удалить`mint.json`файл из вашего репозитория.
    </Note>
  </Accordion>
</AccordionGroup>

### Развертывание ваших изменений

<AccordionGroup>
  <Accordion icon="message-bot" title="Установите наше приложение Github">
    Наше приложение Github автоматически развертывает ваши изменения на вашем сайте документации, поэтому
    вам не нужно управлять развертываниями самостоятельно. Вы можете найти ссылку для установки на
    вашей[панели управления](https://dashboard.mintlify.com). После успешной установки бота
    рядом с хешем коммита репозитория должна появиться галочка.
  </Accordion>

  <Accordion icon="rocket" title="Отправьте ваши изменения">
    [Зафиксируйте и отправьте ваши изменения в
    Git](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)
    чтобы ваши изменения обновились на вашем сайте документации. Если вы отправили изменения и не видите,
    что приложение Github успешно развернуло их, вы также можете вручную
    обновить вашу документацию через нашу[панель управления](https://dashboard.mintlify.com).
  </Accordion>
</AccordionGroup>
