Навигационное меню - это список ссылок на каждом веб-сайте.

Вы, вероятно, будете обновлять `docs.json` каждый раз, когда добавляете новую страницу. Страницы не появляются автоматически.

## Синтаксис навигации

Наш синтаксис навигации рекурсивен, что означает, что вы можете создавать вложенные группы навигации. Вам не нужно включать `.mdx` в названия страниц.

<CodeGroup>
  ```json Regular Navigation
  "navigation": {
    "tabs": [
      {
        "tab": "Docs",
        "groups": [
          {
            "group": "Getting Started",
            "pages": ["quickstart"]
          }
        ]
      }
    ]
  }
  ```

  ```json Nested Navigation
  "navigation": {
    "tabs": [
      {
        "tab": "Docs",
        "groups": [
          {
            "group": "Getting Started",
            "pages": [
              "quickstart",
              {
                "group": "Nested Reference Pages",
                "pages": ["nested-reference-page"]
              }
            ]
          }
        ]
      }
    ]
  }
  ```
</CodeGroup>

## Папки

Просто поместите ваши MDX файлы в папки и обновите пути в `docs.json`.

Например, чтобы иметь страницу по адресу `https://yoursite.com/your-folder/your-page`, вы бы создали папку с названием `your-folder`, содержащую MDX файл с названием `your-page.mdx`.

<Warning>
  Вы не можете использовать `api` для названия папки, если только она не вложена в другую папку. Mintlify использует Next.js, который резервирует папку верхнего уровня `api` для внутренних серверных вызовов. Название папки, такое как `api-reference`, было бы принято.
</Warning>

```json Navigation With Folder
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Group Name",
          "pages": ["your-folder/your-page"]
        }
      ]
    }
  ]
}
```

## Скрытые страницы

MDX файлы, не включенные в `docs.json`, не будут отображаться в боковой панели, но доступны через строку поиска и при прямой ссылке на них.
