## Изображение

### Использование Markdown

Синтаксис [markdown](https://www.markdownguide.org/basic-syntax/#images) позволяет добавлять изображения, используя следующий код

```md
![title](/path/image.jpg)
```

Обратите внимание, что размер файла изображения должен быть менее 5 МБ. В противном случае мы рекомендуем размещать на сервисе, таком как [Cloudinary](https://cloudinary.com/) или [S3](https://aws.amazon.com/s3/). Затем вы можете использовать этот URL и встроить.

### Использование встраиваемых элементов

Для большей настраиваемости изображений вы также можете использовать [встраиваемые элементы](/writing-content/embed) для добавления изображений

```html
<img height="200" src="/path/image.jpg" />
```

### iFrames

Загружает другую HTML-страницу внутри документа. Чаще всего используется для встраивания видео.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
