## Configurez votre développement

Apprenez comment mettre à jour vos documents localement et les déployer publiquement.

### Modifier et prévisualiser

<AccordionGroup>
  <Accordion icon="github" title="Clonez vos documents localement">
    Pendant le processus d'intégration, nous avons créé un dépôt sur votre Github avec
    votre contenu de documentation. Vous pouvez trouver ce dépôt sur notre[tableau de bord](https://dashboard.mintlify.com). Pour cloner le dépôt
    localement, suivez ces[instructions](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    dans votre terminal.
  </Accordion>

  <Accordion icon="rectangle-terminal" title="Prévisualiser les changements">
    La prévisualisation vous aide à vous assurer que vos changements apparaissent comme prévu. Nous avons construit une
    interface en ligne de commande pour rendre ces changements localement.

    1. Installez le[Mintlify CLI](https://www.npmjs.com/package/mintlify)pour prévisualiser les
       changements de documentation localement avec cette commande :`npm i -g mintlify`
    2. Exécutez la commande suivante à la racine de votre documentation (où`docs.json`se trouve) :`mintlify dev`

    <Note>
      Si vous utilisez actuellement le fichier de configuration hérité`mint.json`, veuillez mettre à jour le Mintlify CLI :

      `npm i -g mintlify@latest`
      Et exécutez la nouvelle commande de mise à niveau dans votre dépôt de documentation :

      `mintlify upgrade`
      Vous devriez maintenant utiliser le nouveau fichier de configuration`docs.json`. N'hésitez pas à supprimer le fichier`mint.json`de votre dépôt.
    </Note>
  </Accordion>
</AccordionGroup>

### Déployer vos changements

<AccordionGroup>
  <Accordion icon="message-bot" title="Installez notre application Github">
    Notre application Github déploie automatiquement vos changements sur votre site de documentation, donc vous
    n'avez pas besoin de gérer les déploiements vous-même. Vous pouvez trouver le lien pour l'installer sur
    votre[tableau de bord](https://dashboard.mintlify.com). Une fois que le bot a été
    installé avec succès, il devrait y avoir une coche à côté du hash de commit
    du dépôt.
  </Accordion>

  <Accordion icon="rocket" title="Poussez vos changements">
    [Committez et poussez vos changements vers
    Git](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)
    pour que vos changements soient mis à jour sur votre site de documentation. Si vous poussez et ne voyez pas que
    l'application Github a déployé vos changements avec succès, vous pouvez également mettre à jour
    manuellement votre documentation via notre[tableau de bord](https://dashboard.mintlify.com).
  </Accordion>
</AccordionGroup>
